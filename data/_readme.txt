this folder contains the data for the isimip.org interface

copy contents of sub-folders into respective sub-folders of your local
../climada_data, i.e.

../climada_modules/isimip/data/entities/USA_UnitedStates_Florida_entity.mat to
../climada_data/entities/USA_UnitedStates_Florida_entity.mat

and

../climada_modules/isimip/data/hazards/USA_UnitedStates_Florida_temp_mpi20thcal.mat to
../climada_data/hazards/USA_UnitedStates_Florida_temp_mpi20thcal.mat

and

just copy the whole folder ../climada_modules/isimip/data/isimip to
../climada_data/isimip

see code isimip_step_by_step to get started
